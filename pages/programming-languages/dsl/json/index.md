# JSON - Javascript Object Notation

## JSON 的两种结构
- 键值对的集合：在其他语言中，被叫做object，dictionary，struct，hash table， record等
- 值的有序列表：在其他语言中，被叫做array，vector，list等

## values 的类型
- object
- array
- string
- number
- true/false
- null
