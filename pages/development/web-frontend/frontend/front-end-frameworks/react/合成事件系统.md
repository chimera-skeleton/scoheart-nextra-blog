为什么要费心捕获和冒泡呢？在过去的糟糕日子里，浏览器的交叉兼容性远不如现在，Netscape 只使用事件捕获，而 Internet Explorer 只使用事件冒泡。当 W3C 决定尝试标准化行为并达成共识时，他们最终得到了这个包含两者的系统，这就是现代浏览器所实现的。
