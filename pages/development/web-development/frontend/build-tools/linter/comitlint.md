# commitlint
当我们使用完git commit -m "xxxxxx"后，commitlint可以帮我们检查，我们的提交信息是否符合我们定义的一些规范。

## installation
``` shell
yarn add @commitlint/cli @commitlint/config-conventional
```

## usage
``` shell
touch commitlint.config.js
```
